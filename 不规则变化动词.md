| drive | drove   | driven   | driving  |
| speak | spoke   | spoken   | -        |
| learn | learned | learned  | -        |
| -     | learnt  | learnt   | -        |

learnt v. learn（学习）的过去式和过去分词形式之一。学会，知道，懂得
